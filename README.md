# 🔥 Industrial Safety Monitoring System (ESP32)

## 📌 Overview
This project is an IoT-based Industrial Safety Monitoring System designed to detect hazardous conditions in industrial environments. It continuously monitors parameters like gas leakage, temperature, flame detection, and light intensity.

## 🎯 Objective
To improve industrial safety by providing real-time monitoring and alert mechanisms to prevent accidents.

---

## 🧰 Components Used
- ESP32 Microcontroller
- Gas Sensor (MQ Series)
- Temperature Sensor (DHT11/DHT22)
- Flame Sensor
- LDR Sensor
- Buzzer
- Relay Module (Optional)
- Power Supply

---

## ⚙️ Features
- Real-time monitoring of industrial parameters  
- Gas leakage detection  
- Fire detection using flame sensor  
- Temperature monitoring  
- Automatic alert system (Buzzer / Notification)  
- IoT-based remote monitoring (ESP32 Web Server)  

---

## 🛠️ Technologies Used
- Embedded C  
- Arduino IDE  
- ESP32 WiFi Module  

---

## 🔌 Working Principle
The ESP32 collects data from multiple sensors. If any parameter exceeds the threshold value:
- Buzzer is activated  
- Alert is generated  
- Data is displayed via web server
